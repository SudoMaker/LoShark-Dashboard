# LoShark-Dashboard

Visit [online](https://lsd.sudomaker.com)

## Build

```shell
# CLone the repo
git clone https://github.com/SudoMaker/Loshark-Dashboard

cd Loshark-Dashboard

# Install dependencies
pnpm i

# Build the dashboard
pnpm build

# Preview
pnpm preview
```

## Develop

To start the dev server, run

```shell
pnpm dev
```
